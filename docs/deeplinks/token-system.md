# Deeplink — token system

The credit system that prices optional AI features. Nothing auto-consumes.

## The numbers

- **500 free tokens / month** per user.
- **10€ credit on signup** (card required for pay-per-use).
- **1 token ≈ €0.06–0.07.**

## Fixed vs variable cost features

- **Fixed cost** — a known token price per action (e.g. a single code
  generation). Shown before you confirm.
- **Variable cost** — pay-per-use compute (e.g. an Interceptor run in a headless
  browser). Metered by actual usage.

In both cases the token cost is **shown before execution**. AI calls never
auto-consume — the user always confirms.

## Earning tokens

Tokens are a real, earnable currency:

- **Accepted PRs** to the platform.
- **Published modules** in the block store.
- **Received tips** on your modules.

A community contributor can earn enough tokens to make a living building for the
platform.

## Tips

Tip a module **1 / 10 / 100 tokens** — like a "like" with real value. A signal
plus actual support to the author.

## Principle

Every AI call is labeled and optional. The free, offline core never spends a
token. Tokens only ever pay for our two products (AI-generated code, AI
execution) and the optional wrapper services.
